Guide

How to use DiceBotTK end-to-end

Flow
1

Import accounts from CSV

Go to Accounts → Import CSV/XLSX. Each row is one account with the Dice.fm phone number, email, payment details, proxy, and optional AYCD key.

Expected header row:

phone,email,card_number,exp_month,exp_year,cvc,billing_name,billing_email,billing_phone,billing_postal,billing_country,proxy,aycd_key

Only phone is required. Download the sample from the Accounts page if you want a starting template.

2

Keep the Auth Farm running 24×7

Open the Auth Farm page and hit Start farm. Leave it running continuously — sessions are valid for ~6 days and the farm re-authenticates accounts as they age out.

  • Active sessions are ready to buy.
  • Expiring get rotated automatically.
  • Needs auth go to the front of the queue.

Only accounts with a valid session can run tasks — the farm is what keeps that pool full.

3

Run tasks

Go to the Tasks page. Add one task per account you want to run for a given event, then set the event link, price range, presale code, tier, quantity, and mode.

  • Start an individual task or select multiple and Start / Stop / Edit in bulk.
  • Manual mode reserves the ticket and waits for your approval. Auto purchases immediately.
  • Tasks for accounts without a valid session will refuse to start — auth them first.
